Privacy Policy

The Chrome extension "Daily Adventure" (hereinafter referred to as "this extension") respects user privacy and operates under the following policy.

1. Information Collected

This extension does not collect, store, or transmit personally identifiable information (such as names, email addresses, or browsing history). Any data used is limited strictly to what is necessary for the functionality of the extension.

2. Use of Data

  • Bedtime Settings: The bedtime set by the user is saved locally using chrome.storage.local so that it can be reused after restarting the browser.
  • Notifications: chrome.alarms and chrome.notifications are used to display alerts 30 minutes before the set bedtime.
  • User Interface: Current time and greeting messages are processed and displayed within the extension only.

3. Sharing of Data with Third Parties

This extension does not provide user data to any third parties.

4. Data Retention

Saved data (such as bedtime settings) will be deleted when the user removes it manually or uninstalls the extension.

5. Policy Updates

If future feature updates require handling of new types of data, this privacy policy will be updated accordingly and changes will be clearly stated.
